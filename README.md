## Структура репозитория

```
agents/
  <name>/
    agent-profile.yaml    ← кто этот агент, какая модель, специализация
    skills/               ← подтверждённые паттерны
      <skill-id>.md
    telemetry/
      telemetry.jsonl     ← лог применений: task_type, pattern_used, exchanges, success
spec/
  agent-profile-schema.yaml  ← формальный стандарт полей
```

## Как участнику начать

1. Склонировать репо: `git clone git@github.com:<org>/agentnet-pilot.git ~/agentnet-pilot`
2. Создать папку `agents/<твоё-имя>/`
3. Скопировать `agents/oleg/agent-profile.yaml` как шаблон, заполнить своими данными
4. Добавить в CLAUDE.md блок (см. ниже)
5. Начать записывать телеметрию

## Блок для CLAUDE.md (добавить каждому участнику)

```markdown
## AgentNet (эксперимент, до 2026-03-28)
При начале нетривиальной задачи (>10 мин работы):
1. `git -C ~/agentnet-pilot pull --ff-only` — обновить локальную копию
2. Просмотри `agents/*/skills/*.md` — есть ли паттерн подходящий к задаче?
3. Если да — применй и запиши в лог: "Применён паттерн @автор/название"
4. После задачи: добавь строку в `agents/<твоё-имя>/telemetry/telemetry.jsonl`
```

## Критерий подтверждения гипотезы

- ≥2 из 4 участников применили чужой паттерн (не свой)
- Хотя бы 1 задача: паттерн из сети → измеримо меньше exchanges или выше completion

## Критерий опровержения

- За 4 недели ни один участник не получил реальной пользы от чужого паттерна
