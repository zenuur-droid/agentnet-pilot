## 4. Модель-агностичность: почему это важно сейчас

### Локальные модели — новый фронт

Ollama Modelfile: **3,664** репо. Это люди которые запускают LLM локально (Llama, Qwen, Mistral, DeepSeek).

У локальных моделей нет:
- Persistent memory (вообще)
- Конфигурации ассистента
- Механизма накопления паттернов

**Это чистый рынок.** Нет ни одного инструмента который давал бы им то что AgentNet даёт пользователям Claude Code.

### Что появится в 2026

- **Gemini CLI** (Google уже в бете)
- **GPT-4o в терминале** (OpenAI двигается в эту сторону)
- **Локальные модели на ARM** (Apple Silicon + Snapdragon X Elite): Qwen 2.5, Mistral, DeepSeek R2
- **Специализированные кодинг-модели**: DeepSeek Coder V3 локально, Phi-4

Все они стартуют с нуля каждую сессию. Все будут иметь ту же боль.

**Если AgentNet — протокол, а не продукт под один инструмент**, он работает со всеми из них.

## 5. TAM (Total Addressable Market)

### Текущий (2026)

| Сегмент | Оценка пользователей | Источник |
|---------|---------------------|---------|
| GitHub Copilot | ~1.8M (платящих) | публичные данные GitHub |
| Cursor | ~500K | оценки по funding/growth |
| Claude Code | ~200-500K | оценка по Trends ×419 роста |
| Cline/Windsurf/Aider | ~100K суммарно | GitHub proxies |
| Ollama (локальные) | ~2M+ downloads | публичная статистика |
| **Итого** | **~5-7M** | |

### 2027 (прогноз)

По темпам роста (×4-10 в год для AI tools): **20-50M пользователей** AI coding assistants.

### Кто платит за решение боли

Не все 5-7M. Целевой сегмент:
- Разработчики которые используют AI ассистент >2 часов в день
- Знают о проблеме memory/configuration
- Уже пытались решить сами (CLAUDE.md, .cursorrules и т.д.)

Оценка: **10-15% от TAM** = 500K–1M человек сейчас.

## 6. Конкуренты — честная оценка

| Конкурент | Что делает | Чего не делает |
|-----------|-----------|----------------|
| LangChain Hub | хранит промпты | нет автономного выбора, нет телеметрии |
| OpenAI Memory | помнит пользователя | проприетарно, только GPT, нет паттернов |
| Cursor Rules (community) | люди шарят .cursorrules | ручной процесс, нет effectiveness scores |
| mem0 | персональная память | нет cross-agent sharing |
| PromptLayer | трекинг промптов | enterprise, не для индивидуальных агентов |

**Ниша AgentNet**: открытый, модель-агностичный протокол с объективной телеметрией.
Никто не делает именно это.

## 7. Вывод для питча и стратегии

### Что подтверждено данными

1. **Боль реальна и измерима**: `claude code memory` = 12.4 индекс, `how to use claude code` = 31.7
2. **Рынок растёт**: ×419 за год для Claude Code, остальные также
3. **Боль универсальна**: Cursor, Copilot, Windsurf — одна и та же проблема
4. **1% настраивают, 99% нет**: H-007 подтверждена через selection bias
5. **Локальные модели — следующая волна**: Ollama уже 3.6K репо, в 2026 взрывной рост

### Что меняет позиционирование

Не «инструмент для Claude Code пользователей» →
**«открытый протокол для всех AI coding assistants»**

Аналог: ActivityPub сделал федерацию для соцсетей.
AgentNet делает федерацию для AI агентов — независимо от модели.
