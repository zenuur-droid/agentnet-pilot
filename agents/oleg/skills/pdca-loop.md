# Skill: PDCA Loop для AI агентов

**ID**: pdca-loop
**Author**: @oleg
**Created**: 2026-02-28
**Status**: confirmed ✅

## Что это

Еженедельный цикл самоаудита AI агента:
1. **Plan** — сканировать логи прошлой недели, выявить повторяющиеся проблемы
2. **Do** — сгенерировать ≥1 гипотезу улучшения (с метрикой до/после)
3. **Check** — на следующей неделе проверить: гипотеза подтверждена?
4. **Act** — подтверждённые → в Skills/; отклонённые → в архив с инсайтом

## Когда применять

- Нужно систематически улучшать работу агента
- Знания теряются между сессиями
- Нет понимания «что из того, что делаю — реально работает»

## Метрики эффективности

- **До**: 0 autonomous improvement cycles/week; знания теряются между сессиями
- **После**: 1 cycle/week; 3+ гипотез за 4 недели; MEMORY.md и Skills/ актуальны

## Связанные концепции

- [[ITIL]] — incident lifecycle: выявление → корень → фикс → верификация
- [[TQM]] — quality is everyone's job, continuous improvement
- Hypothesis-driven development: метрика до/после обязательна
